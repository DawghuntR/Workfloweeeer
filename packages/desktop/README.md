# Desktop
Capture desktop workflows with Electron and hotkeys.

---

## Explain
Record desktop steps, screenshots, and metadata.
Provide tray controls, hotkeys, and export actions.

---

## Use
Run the desktop app in development mode.

```bash
npm run dev:desktop
```

---

## Review
Key features:

- Tray menu and global hotkeys
- Fullscreen or window capture scope
- IPC APIs for guide editing and export
- Crash recovery autosave sessions
