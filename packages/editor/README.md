# Editor
Edit steps in a React UI.

---

## Explain
Provide reusable components and state for editing guides.
Manage selections, annotations, and screenshot updates.

---

## Use
Run the editor in dev mode.

```bash
npm run dev:editor
```

---

## Review
Key exports:

- Components: `StepList`, `StepEditor`, `AnnotationEditor`, `ScreenshotReplacer`
- Store: `useEditorStore`
- Hooks: `useKeyboardShortcuts`

---

These components expect core guide types.
